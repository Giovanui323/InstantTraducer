# Incremento Versione e Build del Progetto

Il piano prevede di incrementare la versione del progetto (patch) e avviare il processo di build completo.

## Dettagli Tecnici:
1. **Incremento Versione**: Utilizzerò il comando `npm version patch --no-git-tag-version` per passare dalla versione attuale `3.4.5` alla `3.4.6` nel file [package.json](file:///Users/lucasicignano/Lavoro%20temporaneo/UNIVERSITA/FIVER/gemini-pdf-translator-di-libri-2/package.json).
2. **Esecuzione Build**: Eseguirò il comando `npm run package` (o direttamente lo script `build:all` che combina entrambi i passaggi) per compilare il frontend con Vite e preparare il pacchetto Electron.

## Passaggi Operativi:
1. Eseguire `npm run build:all` nel terminale. Questo comando:
   - Incrementa la versione di una patch.
   - Esegue `vite build`.
   - Genera le icone.
   - Esegue il packaging con `electron-builder`.
