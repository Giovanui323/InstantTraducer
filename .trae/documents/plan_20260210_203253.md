# Piano di Intervento: Performance, Ciclo di Vita e Configurazione AI

## 1. Risoluzione Memory Leak (Rendering PDF)

**Problema:** I log mostrano un degrado delle prestazioni (da 30ms a >1500ms). L'analisi ha confermato che gli oggetti `PDFPageProxy` creati in `src/utils/pdfUtils.ts` non vengono distrutti esplicitamente, causando un accumulo di memoria nella cache interna di PDF.js.
**Soluzione:**

* Modificare `src/utils/pdfUtils.ts` per aggiungere `page.cleanup()` nel blocco `finally` di tutte le funzioni di rendering (`renderDocPageToJpeg`, `renderDocPageToObjectURL`, `renderPageOntoCanvas`, ecc.).

## 2. Eliminazione Rendering Ridondante (Ciclo Infinito)

**Problema:** Le pagine 4, 5, 6, 7, 8 vengono renderizzate ciclicamente senza richiesta utente.
**Causa:** La logica di "prefetch" automatico in `src/App.tsx` accoda le pagine per la traduzione se non sono presenti nella mappa delle traduzioni completate. Tuttavia, la funzione `shouldSkipTranslation` in `useAppTranslation.ts` controlla solo se la traduzione è *completata*, non se è *in corso* o *in coda*. Questo causa un ri-accodamento continuo ad ogni aggiornamento di stato, innescando nuovi tentativi di rendering (per l'acquisizione dell'immagine) che competono con il rendering a schermo.
**Soluzione:**

* Aggiornare `shouldSkipTranslation` in `src/hooks/useAppTranslation.ts` per restituire `true` anche se la pagina è in stato `loading`, `processing` o presente nella coda attiva.

## 3. Gestione Asset Orfani (Sicurezza Salvataggio)

**Problema:** Le immagini vengono salvate su disco anche quando il progetto è in fase di rinomina o blacklist (es. durante un cambio ID), creando file orfani.
**Causa:** Le chiamate a `window.electronAPI.saveProjectImage` (IPC) bypassano i controlli di sicurezza presenti in `useAppLibrary` per il file JSON.
**Soluzione:**

* Esporre la funzione `isBlocked(fileId)` da `src/hooks/useAppLibrary.ts`.

* In `src/App.tsx`, intercettare tutte le chiamate di salvataggio immagini (inclusa `saveSourceForPage` passata all'executor) e bloccarle se l'ID è in blacklist o transizione.
